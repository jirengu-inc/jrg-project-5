JS 文件已经可以运行了，Vue 也引入了，那我们就直接开始撸代码了。

期间会有一些你不懂的东西，比如 Vue 组件、Vue 事件绑定、双向绑定等。

遇到不懂的东西，你应该用「copy - run - modify」的套路把官网上的例子运行一遍，然后改点东西。只需要知道你改一个东西会有什么作用就可以了。
